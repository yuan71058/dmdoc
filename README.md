# 大漠插件 API 文档

> 473 个接口完整说明，一站式 API 查询平台

## 快速开始

访问在线文档：https://yuan71058.github.io/dmdoc/

## 项目介绍

本项目是将大漠插件 CHM 帮助文档转换为现代化 Web 文档站点，使用 Docsify 构建。

## 功能特性

- ✅ 完整的 API 文档（473 个接口）
- ✅ 快速搜索功能
- ✅ 响应式设计（支持移动端）
- ✅ 代码高亮显示
- ✅ 自动部署到 GitHub Pages

## 技术栈

- **Docsify** - 文档生成器
- **GitHub Pages** - 静态托管
- **GitHub Actions** - 自动部署
- **Prism.js** - 代码高亮

## 本地运行

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx serve
```

然后访问 http://localhost:8000
