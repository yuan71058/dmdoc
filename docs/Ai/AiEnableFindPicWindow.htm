й–ёжҲЈе§ӨйҸҶзҶәзІ»йҲ§о„ҒзҰ’й”ҹпҝҪ:

йҗ дҪёеўҪйҗӨе—ӣеј°йЎ–ж°ӯе„Ҡй–ёпёәеҠҸйҗЁзҶјжӮҪй”ҹпҝҪAiFindPicXXзјҒоҲқоҮ§йҚЁоҒҲе№’йҠүгғҘз¶“й–әеҶҝжӢ·,й–әеӢөеһ°йҺҜдҪёоҮЈйҗҹжқҝжҜүй–№еһ«е„іеЁҙжЁјзҙ’йҺҫеҜёдәҜй–»г„Ҙе«®йҗӣгғ©еҙЈй”ҹпҝҪ.  й–әеӮһвӮ¬ж¶ҡв”’йҗ е¬Әе•ійҗҰпҝҪ. е§’ж¶ҷЗ№йЎ“е©ҡеј°йЎ–ж°¬еҪ й—ӮеӮӨеҺҫеЁҲпҝҪ.

й–ёжҲЈе§ӨйҸҶзҶјеҙўйҗ”йЈҺвӮ¬пҝҪ:

long AiEnableFindPicWindow(enable)

й–ёж¬ҸеҖ№йҸҶзҶәвӮ¬и§„зҹ®з» пҝҪ:

enable й–әдҪ№жҪҷй‘ёдјҙеј«й”ҹпҝҪ: 0 й–ёеҝ”еЁҠеҰ«пҝҪ
1 зҖөоҶ№еҒ“й–ёж°ҫжӢ·

йҸүв•ӮжҹЁеЁІж ӯеҙҗй”ҹпҝҪ:

й–әдҪ№жҪҷй‘ёдјҙеј«й”ҹпҝҪ:
0: е©ўжғ°зІҫз‘ҷпҝҪ
1: й–№еӯҳеҠ•жҝ®пҝҪ

set dm = CreateObject("dm.dmsoft")

dm_ret = dm.SetDisplayInput("screen")
