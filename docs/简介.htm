🚀 大漠插件 API 文档

关于大漠插件

大漠插件是一款功能强大的 Windows 自动化测试工具，提供丰富的接口用于窗口操作、图色识别、文字识别、内存读写等功能。 它广泛应用于游戏外挂、自动化测试、机器人开发等领域。

📚 主要功能模块

💡 Ai 模块

基于 AI 的图片识别功能，支持 YOLO 目标检测，提供更智能的图片匹配和识别能力。

🎨 图色模块

强大的颜色查找、图片查找功能，支持透明色、多点找图等高级特性。

💾 内存模块

进程内存读写、搜索功能，支持整数、浮点数、字符串等多种数据类型。

🖥️ 窗口模块

窗口枚举、查找、控制功能，可获取窗口信息、发送消息、调整窗口状态。

⌨️ 键鼠模块

模拟键盘按键和鼠标操作，支持后台模式，不影响用户正常使用电脑。

🔤 文字识别

OCR 文字识别功能，支持自定义字库，可识别屏幕上的文字内容。

🛡️ 防护盾

反调试、反注入保护功能，增强程序的自我保护能力。

⚙️ 后台设置

绑定窗口、后台运行配置，实现真正的后台自动化操作。

🎯 核心特性

  • 后台模式 - 不干扰用户正常使用，可在后台执行自动化任务
  • 多语言支持 - 支持 C++、C#、VB、Python 等多种编程语言调用
  • 高效稳定 - 底层驱动实现，性能优异，运行稳定
  • 功能全面 - 覆盖窗口、图色、内存、文字识别等全方位自动化需求
  • 易于使用 - 接口简单直观，文档详细，快速上手

💻 代码示例

以下是简单的 VBScript 示例代码：

' 绑定桌面窗口
hwnd = dm.BindWindow("自然模式", "前台", "桌面", "桌面", 0)

' 查找图片
result = dm.FindPic(0, 0, 1920, 1080, "button.bmp", "0.9", intX, intY)

' 模拟点击
if intX >= 0 then
    dm.MoveTo(intX, intY)
    dm.LeftClick()
end if

' 文字识别
text = dm.FindStr(0, 0, 1920, 1080, "确定")

' 解绑
dm.UnBindWindow()
