й–ёжҲЈе§ӨйҸҶзҶәзІ»йҲ§о„ҒзҰ’й”ҹпҝҪ:

й–әеұ»е„Іжҫ№жЁјзІӯй”•вӮ¬йҺ®еәЈзҢҫз’ҮеҸүеҖій–№еӯҳзүһйҲ§о„ҖжҮҸйҗ–пҪҶпјҗеҰҜзғҳеҖій–»г„Ҙе«°йҠҶе©„д»ҰйҚҢж°¬и®Ійҗҹж¬Һиғ¶йҗӣгғ©еҙЈй”ҹпҝҪ,е©өвҖіеҖ№йҗҸеӨҗе№җйҚҘС…жҡ°еЁҙж»җжӢ·parent,й–ёжҺ“з‘Ҙе©ҖпҝҪparentй–»г„Ҙе«®йЎ‘еӣЁзЁүйҲ§о„Ӯд»ҰйҚҢж°ұж‘ҚзјҒж„ӯоҡ…иӨ°ж¶ҷзЁүйЎ“зҶәеҸҖй–№зўүжӢ·.

й–ёжҲЈе§ӨйҸҶзҶјеҙўйҗ”йЈҺвӮ¬пҝҪ:

long FindWindowEx(parent,class,title)

й–ёж¬ҸеҖ№йҸҶзҶәвӮ¬и§„зҹ®з» пҝҪ:

parent й–әдҪ№жҪҷй‘ёдјҙеј«й”ҹпҝҪ: й–»ж §еўҺйҗӣгғ©еҙЈй”қе‘ӯз¶–й–әеұҪе«ёз»ұжҝҮдҝҠйҚҢж¶ҡдәҜеЁ‘ж’№ж«ЈйҲ№ж ӯж•ҚзҒһи—үзҒҹй–ёж §з¶ҠйҚҳгӮүе№ҚйҲ§о„„ејөжҫ¶ж„©вӮ¬е©„д»ҰйҚҢж»…еҙ¶й–ёж¬ҷжӢ·

 

class йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: зјҒж„ӯоҡ…иӨ°ж¶ҡзҢҫз’ҮеҸүеҖій–ҝж¶іжӢ·е©өвҖіеҖ№йҗҸеӨӢзЁүй“Ҹе…үж•„й–ҝж¶іжӢ·й–ёжҺ“з‘ҘзҒҸоҮҖжҹҠе®Ҙе——о•Ій–әе ¬жӢ·. йҸүв•Ӯз‘©йҚЈзғҪжғғйҚ•оӮўзҲұй—Ғжқҝз§өеҰІзЁҝОҹй”ӣеӢӯЛҮй–ёж §з¶ҠйҚҳпҝҪ.

title йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: зјҒж„ӯоҡ…иӨ°ж¶ўејҪйҚҘо„үжҡҜ,е©өвҖіеҖ№йҗҸеӨӢзЁүй“Ҹе…үж•„й–ҝж¶ҳиӢҜйҚЁоҲһеҙ иӨ°жҺ‘еёій–№зўҳеҒ“й–әе ¬жӢ·. йҸүв•Ӯз‘©йҚЈзғҪжғғйҚ•оӮўзҲұй—Ғжқҝз§өеҰІзЁҝОҹй”ӣеӢӯЛҮй–ёж §з¶ҠйҚҳпҝҪ.

йҸүв•ӮжҹЁеЁІж ӯеҙҗй”ҹпҝҪ:

й–әдҪ№жҪҷй‘ёдјҙеј«й”ҹпҝҪ:
й–әдҪ№жҪҷй‘ёдјҙеј«йҺ·еұ»вӮ¬еҶӘзІҲй“Ҹи§„з•ұзјҒж„ӯоҡ…иӨ°ж¶ўеҙЈйҠүпёҫеҠҙй–ҝж¶ҳжң¬йҗҘе‘ҙе№ҚйҺҜСғз…ӮйҸүв•ӮжҹЁеЁІпҝҪ0
