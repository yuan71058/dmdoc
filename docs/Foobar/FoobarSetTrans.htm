й–ёжҲЈе§ӨйҸҶзҶәзІ»йҲ§о„ҒзҰ’й”ҹпҝҪ:

йҗ дҪёеўҪйҗӨе—ӣе№җйҚҘС…жҡ°FoobarзјҒж„ӯоҡ…иӨ°ж¶ўжғғйҚ•о„ҒРҰй–ёж°ҫз®“йҲ§о„Җз№‘еҰІпҝҪ

й–ёжҲЈе§ӨйҸҶзҶјеҙўйҗ”йЈҺвӮ¬пҝҪ:

long FoobarSetTrans(hwnd,is_trans,color,sim)

й–ёж¬ҸеҖ№йҸҶзҶәвӮ¬и§„зҹ®з» пҝҪ:

hwnd й–әдҪ№жҪҷй‘ёдјҙеј«й”ҹпҝҪ: й–№зЁҝжҙӨйҗЈй№ғжғғй”ҹпҝҪFoobarзјҒж„ӯоҡ…иӨ°ж¶ўеҙЈйҠүпёҫеҠҙ,жҝ®жҝ„еҖ•иӨ°зӮәеј»йҚ•оӮӨз®‘еҰһгӮҮзІҜеҰІжҗҒжҹ…е§ҳеҙҮз®–CreateFoobarxxxй–ёжҺ“зЁ‘зјӮж’»жҮ“зҒһзӮ¬йҷҚ

is_trans й–әдҪ№жҪҷй‘ёдјҙеј«й”ҹпҝҪ: й–әеӢөеһ°йҺҜдҫҖжҹ…и№Үж—Җоқ–. 0еЁ‘ж’№жЎЁз»—еӨҗжҹ…и№Үж—Җоқ–(жҝ®жҝ„еҖ–еҰһпҝҪ,colorй–ёж»җжӢ·simй–әеҶӘе§өйҸ…пҝҪ)й–ҝж¶іжӢ·1еЁ‘ж’ҙжӨҪйҲ§о„Җз№‘еҰІпҝҪ.

color йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: й—ҒоӮўз№‘еҰІжҲҰжҮқй”ҹпҝҪ(RRGGBB)

sim й–ёж¬Ҹзһјз»ЁжҢҺжғ”й””еҖҹзҷҒй–»ж„ҜоқүйҸҶпҝҪ: й—ҒоӮўз№‘еҰІжҲҰжҮқй‘јо„Ғз•ұй–»в•…жҷІйҺ¶вӮ¬й–ёе©ҸжӢ· 0.1-1.0

йҸүв•ӮжҹЁеЁІж ӯеҙҗй”ҹпҝҪ:

й–әдҪ№жҪҷй‘ёдјҙеј«й”ҹпҝҪ:
0 : е©ўжғ°зІҫз‘ҷпҝҪ

1 : й–№еӯҳеҠ•жҝ®пҝҪ

зјҒвӮ¬жқһжү®дјҗ:

foobar=dm.CreateFoobarRoundRect(hwnd,1,1,300,300,100,100)

dm_ret = dm.FoobarSetFont(foobar,"йҗҺз‘°оғ„зјҚпҝҪ",50,0)

dm.FoobarSetTrans foobar,1,"000000",1.0

do

   dm_ret = dm.FoobarFillRect(foobar,0,0,300,300,"000000")

   dm_ret = dm.FoobarDrawText(foobar,0,0,300,100,"жҝһжқ‘оғҲйҗҰпҝҪ","FF0000",1)

   dm.foobarupdate foobar

   delay 100

Loop

EndScript
