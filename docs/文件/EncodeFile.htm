й–ёжҲЈе§ӨйҸҶзҶәзІ»йҲ§о„ҒзҰ’й”ҹпҝҪ:

й–ёж—Ӯе§ҙйҗҰжҲҰе№җйҚҘС…жҡ°й–»г„Ҙе«ӯйҸӢеҶ©зҰ’й”ҹпҝҪ.

й–ёжҲЈе§ӨйҸҶзҶјеҙўйҗ”йЈҺвӮ¬пҝҪ:

long EncodeFile(file,pwd)

й–ёж¬ҸеҖ№йҸҶзҶәвӮ¬и§„зҹ®з» пҝҪ:

file йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: й–әеӮҡжҙЈеЁҶгҲӨеҙҘй”ҹпҝҪ.

pwd йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: йҗҺйқӣжЈӣйҗ–пҝҪ.

йҸүв•ӮжҹЁеЁІж ӯеҙҗй”ҹпҝҪ:

й–әдҪ№жҪҷй‘ёдјҙеј«й”ҹпҝҪ:
0 : е©ўжғ°зІҫз‘ҷпҝҪ
1 : й–№еӯҳеҠ•жҝ®пҝҪ

зјҒвӮ¬жқһжү®дјҗ:

// зјӮдҪ№з№ӮйЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
dm.EncodeFile "c:\test_game\cfg.ini","1234"

// й–»в•ғЗ№йЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
dm.SetPath "c:\test_game"
dm.EncodeFile "1.bmp","1234"

е©өвҖіеҖ№йҗҸеӨҠоқўйҠҲе—ҳзҖ®еЁҙзҠәжіӣйҚ‘пј„зҙ’и№Үж’іоқЈйҗҺйқӣжЈҙз»ұжҝҲжӢ«йҚҗоҒ…жҡҸжҝ®жҝ„еҖ•йҚӨйҒҚеј«жЈ°дҪәз‘қеЁҙе…јзўҚе©ҖдҪ№зҰ’з’ҒеҙҮз§ҝй–әдҪёзүҠйҗҸпҝҪ.
