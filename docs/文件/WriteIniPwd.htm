й–ёжҲЈе§ӨйҸҶзҶәзІ»йҲ§о„ҒзҰ’й”ҹпҝҪ:

й–ёж°ӯеһ¶зҖөж°ұвӮ¬и§„ж°ЁеЁҲпҝҪIniй–ёж„ӯз‘ҘйҚҷе—ҳз©ұйҲ©еҶ§зҙ–.й–әвӮ¬йЎ–ж¶ҳзҳ®й–ёж—Ӯе§ҙйҗҰжҲҰеј¬йҚҘпёҪоӮҪ

й–ёжҲЈе§ӨйҸҶзҶјеҙўйҗ”йЈҺвӮ¬пҝҪ:

long WriteIniPwd(section,key,value,file,pwd)

й–ёж¬ҸеҖ№йҸҶзҶәвӮ¬и§„зҹ®з» пҝҪ:

section йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: йҗҸеҝ“з№—жҝЎо…ўеҙҘй”ҹпҝҪ

key йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: й–ёж¬җп№ўйҚЈжҙӘеҙҘй”ҹпҝҪ.

value йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: й–ёж¬җп№ўйҚЈжҙӘеҙҳйҚӣо…§е•Ү

file йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: iniй–әеӮҡжҙЈеЁҶгҲӨеҙҘй”ҹпҝҪ.

pwd йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: йҗҺйқӣжЈӣйҗ–пҝҪ.

йҸүв•ӮжҹЁеЁІж ӯеҙҗй”ҹпҝҪ:

й–әдҪ№жҪҷй‘ёдјҙеј«й”ҹпҝҪ:
0 : е©ўжғ°зІҫз‘ҷпҝҪ
1 : й–№еӯҳеҠ•жҝ®пҝҪ

зјҒвӮ¬жқһжү®дјҗ:

// зјӮдҪ№з№ӮйЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
dm.WriteIniPwd "Global","var1","123","c:\test_game\cfg.ini","1234"

// й–»в•ғЗ№йЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
dm.SetPath "c:\test_game"
dm.WriteIniPwd "Global","var1","123","cfg.ini","1234"

жҝһеӨӣжӢ· : жҝ®жҝ„еҖ•йҚӨйҒҚеј«йҗ—е ҹРҰе©ўиҲөж°ЁйҚӨеәЈзІҷзҖЈоҒҶжҡ”й–ёеҝӢеҠҺеЁҲпҝҪ. е©ўиҲөж°ЁйҚӨеәЈзІҷзҖЈоҒ…еҖұй–әеҶ©е„јйЎ•дјҙеҙҳеЁҶеҝ”еҖұеЁ‘ж“ғдәқйҸӢеҶ©зҰ’йҸҲз”өз‘қеЁҙе…јеІёйҲ§о„Ӯе§өйҚЁж°¶еј¬йҚҘпёҪоӮҪй—ҒжҢҺз‘Өз’җпҝҪ. еЁҙпҪ…жЈҷеҰІе‘Ҡеҫ„е§ҳеҙҮз®»зјҒе¬ӘоғҶеҰІе‘ҠзЁүе®Ҙе‘Ҝжҡ”й–ёеҝӢеҠҺеЁҲпҝҪ,йҗҹжӣҡз·ҡжөјв•…еҙ—е®Ҙе’ҒжЁӢйҸүв•ӮзЁ“йҲ»еҘёеҙҘзҒһзӮ¬оҳ§еЁҙпҪҲжі•йҸҒгӮ…оқўйҠҲе——еӨҚй–ёж¬ҷжӢ·,й–ёж°ҫз®‘йҚЁоҲӣеҜје§ҳеһӣеҒ“йҗҳеҶІзҒҮй–әдҪәеў—е®“дҫҖжҹЁеЁҶеәӨзЎҖ.

е©өвҖіеҖ№йҗҸеӨҠоқўйҠҲе—ҳзҖ®еЁҙзҠәеҹ–йҗҘе‘ҙеҙқйҗҳиҢ¬ж§•й–ҝж¶ҳзҹҒйҗЁзҶјжӮҪйҠҠпё»еҠғй–ёжҲЈе§ӨйҸҶзҶёеҜје§ҳеҶІж®°й–ёж–»еҠҢжҝ®зӮөвӮ¬зўүжӢ·.
