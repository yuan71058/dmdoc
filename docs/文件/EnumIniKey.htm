й–ёжҲЈе§ӨйҸҶзҶәзІ»йҲ§о„ҒзҰ’й”ҹпҝҪ:

й–әе¶Ҳоқүе®“дҫҖе№җйҚҘС…жҡ°й–»оҹ’жӢ·iniй–әеӮҡжҙЈеЁҶгҲЎзҰ’йҠүгғҘжҢӨsection,й–әе¬«зҹ®жҝЎеӣ§оқўй”ҹпҝҪsectionеЁ‘ж“ғеҺҪжҫ§е¶үејөжҫ¶еұҫз•ұkeyй–ёж°ҫжӢ·

й–ёжҲЈе§ӨйҸҶзҶјеҙўйҗ”йЈҺвӮ¬пҝҪ:

string EnumIniKey(section,file)

й–ёж¬ҸеҖ№йҸҶзҶәвӮ¬и§„зҹ®з» пҝҪ:

section йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: йҗҸеҝ“з№—жҝЎо…ўеҙҘй”ҹпҝҪ. (еЁ‘ж’із§ҙиӨ°еҸүзЁүй“Ҹе…үж•„)
file йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: iniй–әеӮҡжҙЈеЁҶгҲӨеҙҘй”ҹпҝҪ.

йҸүв•ӮжҹЁеЁІж ӯеҙҗй”ҹпҝҪ:

йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ:
жҝ®пҪ…з№Ӣй–ІпҝҪkeyй–»оқ®жӢ·"|"й–әеӨҲе„ҙз»»ж¶ўе№’йҠүоҲңзӨүе©өвҖіеҖ№йҗҸеӨҠв–ҪйҲ©еҶ©з®’keyй–ҝж¶ҳиӢҜйҚЁоҲӣжҪ»йҸӮжҢҺзӨҖзјҒеұҪжҗ«йҗЎС…зІӯй””иғҜоӣ«. жҝ®пҪҶжҹЁйЎ©пҝҪ"aaa|bbb|ccc"

зјҒвӮ¬жқһжү®дјҗ:

// зјӮдҪ№з№ӮйЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
dm_ret = dm.EnumIniKey("aaa","c:\test_game\cfg.ini")

// й–»в•ғЗ№йЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
dm.SetPath "c:\test_game"
dm_ret = dm.EnumIniKey("aaa","cfg.ini")

if len(dm_ret) > 0 then
keys = split(dm_ret,"|")

й–ёж¬ҷз®‘йЎҰпҝҪ,жҝ®жҝ„еҖ•йҚӨйҒҚеј«йҗ—е ҹпҝҘжҝһеӨӢжҙҳйҗҸеӣЁзЁүйҗҗо„Ӯжўҫй–әе ¬жӢ·sectionй–»оҹ’жӢ·key.
