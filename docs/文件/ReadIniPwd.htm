й–ёжҲЈе§ӨйҸҶзҶәзІ»йҲ§о„ҒзҰ’й”ҹпҝҪ:

еЁҙзӢ…жӢ·IniеЁ‘ж“ғеҸҖйЎ•дјҙеҙЈйҸҚг„Ҙзҳ№йҗҺи§„зҹ®ж·Үе©Үе№Ғй”ҹпҝҪ.й–ёж¬ҳеһұйҸҒоҮҖе№җжөЈзЁҝоқЈйҗҺйқӣжЈҷйҸӢеҶ©зҰ’й”ҹпҝҪ  

й–ёжҲЈе§ӨйҸҶзҶјеҙўйҗ”йЈҺвӮ¬пҝҪ:

string ReadIniPwd(section,key,file,pwd)

й–ёж¬ҸеҖ№йҸҶзҶәвӮ¬и§„зҹ®з» пҝҪ:

section йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: йҗҸеҝ“з№—жҝЎо…ўеҙҘй”ҹпҝҪ

key йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: й–ёж¬җп№ўйҚЈжҙӘеҙҘй”ҹпҝҪ.

file йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: iniй–әеӮҡжҙЈеЁҶгҲӨеҙҘй”ҹпҝҪ.

pwd йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: йҗҺйқӣжЈӣйҗ–пҝҪ

йҸүв•ӮжҹЁеЁІж ӯеҙҗй”ҹпҝҪ:

йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ:
йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүйҺ»жҺ•еҹҢзҖөоҶјз№—йҠҶеҶ©жҪҸйҚ“С„з•ұйҗ еӣӘоҮ§иӨ°еӣ¬еҙҡйҺөоҶҪз•ұй–ёж„¬жҮҺйЎ”пҝҪ

зјҒвӮ¬жқһжү®дјҗ:

// зјӮдҪ№з№ӮйЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
Text = dm.ReadIniPwd("Global","var1","c:\test_game\cfg.ini","1234")

// й–»в•ғЗ№йЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
dm.SetPath "c:\test_game"
Text = dm.ReadIniPwd("Global","var1","cfg.ini","1234")

жҝһеӨӣжӢ· : жҝ®жҝ„еҖ•йҚӨйҒҚеј«йҗ—е ҹРҰе©ўиҲөж°ЁйҚӨеәЈзІҷзҖЈоҒҶжҡ”й–ёеҝӢеҠҺеЁҲпҝҪ. е©ўиҲөж°ЁйҚӨеәЈзІҷзҖЈоҒ…еҖұй–әеҶҝжӢ·йҗ еӣӘоҮ§йҚҹж’»еҙҘзҒһеҘёеҡӢй–әеӮҡжҙЈеЁҶпҝҪеЁ‘ж’із§із»ұдјҙжҹ…йҗҳеҶІзҒҮй–әеӮҡжҙЈеЁҶгҲӨжҹЁеЁҶеәӨзЎҖ. еЁҙпҪ…жЈҷеҰІе‘Ҡеҫ„е§ҳеҙҮз®»зјҒе¬ӘоғҶеҰІе‘ҠзЁүе®Ҙе‘Ҝжҡ”й–ёеҝӢеҠҺеЁҲпҝҪ,йҗҹжӣҡз·ҡжөјв•…еҙ—е®Ҙе’ҒжЁӢйҸүв•ӮзЁ“йҲ»еҘёеҙҘзҒһзӮ¬оҳ§еЁҙпҪҲжі•йҸҒгӮ…оқўйҠҲе——еӨҚй–ёж¬ҷжӢ·,й–ёж°ҫз®‘йҚЁоҲӣеҜје§ҳеһӣеҒ“йҗҳеҶІзҒҮй–әдҪәеў—е®“дҫҖжҹЁеЁҶеәӨзЎҖ.

е©өвҖіеҖ№йҗҸеӨҗеј¬йҚҘпёҪоӮҪжҝһеұёвӮ¬еҶІоқЈйҗҺйқӣжЈҙз»ұжҝҮзЁҠйҗ”еі°и®ІеЁҙзҠ®е„ІйЎ’ж»…ж•®е§қеІҠеҡўй–ёж¬ҷжӢ·.
