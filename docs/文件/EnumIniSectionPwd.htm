й–ёжҲЈе§ӨйҸҶзҶәзІ»йҲ§о„ҒзҰ’й”ҹпҝҪ:

й–әе¶Ҳоқүе®“дҫҖе№җйҚҘС…жҡ°й–»оҹ’жӢ·iniй–әеӮҡжҙЈеЁҶпҝҪ,й–әе¬«зҹ®жҝЎеӣ§оқўй”ҹпҝҪiniеЁ‘ж“ғеҺҪжҫ§е¶үејөжҫ¶еұҫз•ұSection(йҗҸеҝ“з№—жҝЎо…ўеҙҘй”ҹпҝҪ) й–ёж¬ҳеһұйҸҒоҮҖе№җжөЈзЁҝоқЈйҗҺйқӣжЈҷйҸӢеҶ©зҰ’й”ҹпҝҪ

й–ёжҲЈе§ӨйҸҶзҶјеҙўйҗ”йЈҺвӮ¬пҝҪ:

string EnumIniSectionPwd(file,pwd)

й–ёж¬ҸеҖ№йҸҶзҶәвӮ¬и§„зҹ®з» пҝҪ:

file йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: iniй–әеӮҡжҙЈеЁҶгҲӨеҙҘй”ҹпҝҪ.
pwd йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ: йҗҺйқӣжЈӣйҗ–пҝҪ

йҸүв•ӮжҹЁеЁІж ӯеҙҗй”ҹпҝҪ:

йҗҺж¶ҷоҡҲйЎ‘дҪ№зЁүй”ҹпҝҪ:
жҝ®пҪ…з№Ӣй–Іж»…дәёи№Үж°¬Оқй–ёж°¬з§¶йҸҒпҝҪ"|"й–әеӨҲе„ҙз»»ж¶ўе№’йҠүоҲңзӨүе©өвҖіеҖ№йҗҸеӨҠв–ҪйҲ©еҶ©з®’йҗҸеҝ“з№—жҝЎо…ўж•ҚзҒһи—үзҒҹйҸүв•ӮжҹЁеЁІж «зІҡйҚқеӢӯж‘ҹзјҒжҘӢзјҡз‘•пҝҪ. жҝ®пҪҶжҹЁйЎ©пҝҪ"aaa|bbb|ccc"

зјҒвӮ¬жқһжү®дјҗ:

// зјӮдҪ№з№ӮйЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
dm_ret = dm.EnumIniSectionPwd("c:\test_game\cfg.ini","123")

// й–»в•ғЗ№йЎ•оҶҫжҚ„йЎ–ж°ұзӘһ
dm.SetPath "c:\test_game"
dm_ret = dm.EnumIniSectionPwd("cfg.ini","123")

if len(dm_ret) > 0 then
sections = split(dm_ret,"|")
count = ubound(sections) + 1
index = 0
Do While index < count
     TracePrint sections(index)
     index = index + 1
Loop

end if

жҝһеӨӣжӢ· : жҝ®жҝ„еҖ•йҚӨйҒҚеј«йҗ—е ҹРҰе©ўиҲөж°ЁйҚӨеәЈзІҷзҖЈоҒҶжҡ”й–ёеҝӢеҠҺеЁҲпҝҪ. е©ўиҲөж°ЁйҚӨеәЈзІҷзҖЈоҒ…еҖұй–әеҶ©е„јйЎ•дјҙеҙҳеЁҶеҝ”еҖұеЁ‘ж“ғдәқйҸӢеҶ©зҰ’йҸҲз”өз‘қеЁҙе…јеІёйҲ§о„Ӯе§өйҚЁж°¶еј¬йҚҘпёҪоӮҪй—ҒжҢҺз‘Өз’җпҝҪ. еЁҙпҪ…жЈҷеҰІе‘Ҡеҫ„е§ҳеҙҮз®»зјҒе¬ӘоғҶеҰІе‘ҠзЁүе®Ҙе‘Ҝжҡ”й–ёеҝӢеҠҺеЁҲпҝҪ,йҗҹжӣҡз·ҡжөјв•…еҙ—е®Ҙе’ҒжЁӢйҸүв•ӮзЁ“йҲ»еҘёеҙҘзҒһзӮ¬оҳ§еЁҙпҪҲжі•йҸҒгӮ…оқўйҠҲе——еӨҚй–ёж¬ҷжӢ·,й–ёж°ҫз®‘йҚЁоҲӣеҜје§ҳеһӣеҒ“йҗҳеҶІзҒҮй–әдҪәеў—е®“дҫҖжҹЁеЁҶеәӨзЎҖ.

е©өвҖіеҖ№йҗҸеӨҗеј¬йҚҘпёҪоӮҪжҝһеұёвӮ¬еҶІоқЈйҗҺйқӣжЈҙз»ұжҝҮзЁҠйҗ”еі°и®ІеЁҙзҠ®е„ІйЎ’ж»…ж•®е§қеІҠеҡўй–ёж¬ҷжӢ·.
