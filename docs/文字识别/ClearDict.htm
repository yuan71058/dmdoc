函数简介:

清空指定的字库.

函数原型:

long ClearDict(index)

参数定义:

index 整形数:字库的序号,取值为0-99,目前最多支持100个字库

返回值:

整形数:
0:失败
1:成功

示例:


dm.ClearDict 0

 

注意: 此函数尽量在小字库中使用，大字库中使用AddDict速度比较慢

另外，此函数支持清空内存中的字库，而不是字库文件本身.
